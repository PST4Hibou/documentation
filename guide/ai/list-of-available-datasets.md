# List of Datasets

We store all datasets on the plateform [HuggingFace](https://huggingface.co) on the [Hibou-Foundation profile](https://huggingface.co/Hibou-Foundation)

Here are the one we use the most for training and evaluating our models:



| Name                                                     | Description                                              | Number of rows      |
|----------------------------------------------------------|----------------------------------------------------------|---------------------|
| Hibou-Foundation/all_tests_ds_3                          | Raw audio, chunked of parabolics/non parabolics tests    | 12,987              |
| Hibou-Foundation/parabolics_sound_ds                     | Raw audio of parabolics sounds only                      | 1336                |
| Hibou-Foundation/ds_121_melspecto_noaug_balanced_chunked | MelSpectrogram chunked and balanced                      | 439,174             |
| Hibou-Foundation/ds_10_raw_aug_0-5_balanced_chunked      | Raw audio with a 0.5 augmentation of data and balanced   | 614,819             |
| Hibou-Foundation/big_ds_5_specto_noaug_chunked_balanced  | Spectogram and balanced                                  | 439,174             |
| Hibou-Foundation/big_ds_4_raw_wav_balanced               | Raw audio and balanced                                   | 439,174             |
| Hibou-Foundation/big_ds_raw_wav                          | Raw only                                                 | 774,709             |
| Hibou-Foundation/big_ds_preprocessed_specto_augment_1    | Spectogram and a lot of augmentation                     | N/A (135GB of data) |
| n1coc4cola/maotouying                                    | It contains multiple datasets of audio drone and others. | N/A (135GB of data) |
